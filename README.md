フォロバさんを探せで作ってきたソフトがそれなりに形になってきたので公開します。
できることは以下の通りです。

- 自動フォロー
- 自動フォロー解除
- フォロー対象の自動検索
- 自動ツイート


## 概要

名称：Twapi

ツイッターのツイートやフォローを自動化することを目的としています。
